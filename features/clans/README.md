# Feature: Clans / Clanes del Barrio

**Status:** spec  
**Phase:** alpha  
**Owner:** `game_designer` + `technical_director`  
**Docs:** [CLAN_SYSTEM_GUIDE](../../docs/game/CLAN_SYSTEM_GUIDE.md), [ECONOMY_GUIDE §9](../../docs/systems/ECONOMY_GUIDE.md), [PROGRESSION_GUIDE §7](../../docs/game/PROGRESSION_GUIDE.md)

## Resumen

Grupos de chicos del barrio: **membresía**, **identidad visual**, **banco compartido** y **proyectos cooperativos**. Pilar Pertenencia — opcional pero valioso.

## MVP scope (alpha)

- [ ] Crear/unirse a clan (max 12 miembros)
- [ ] Roles: Fundador, Capitán, Miembro
- [ ] Banco clan (chapitas + materiales donados)
- [ ] 1 proyecto clan starter → pieza especial
- [ ] Bandera visible en carrera libre

## Out of scope (alpha)

- Chat clan / reportes
- Disolver clan con confirmación legal
- Ranking global cross-barrio

## Dependencias

- `economy` — banco, donaciones, entry fees
- `race` — carreras clan, bonus tag
- `events` — torneos y carreras clan vs clan
- Assets: banderas clan, hub POI
