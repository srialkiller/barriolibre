# Events — QA

## Criterios de aceptación

| ID | Criterio | Prioridad |
|---|---|---|
| EV-QA-01 | Evento desbloqueado aparece en calendario | P0 |
| EV-QA-02 | Inscripción debita entry fee si aplica | P0 |
| EV-QA-03 | Start evento lanza race/clan flow correcto | P0 |
| EV-QA-04 | Evento completado marca slot como done | P0 |
| EV-QA-05 | Evento bloqueado muestra requisito rep/carrera | P1 |
| EV-QA-06 | Calendario persiste entre sesiones | P1 |

## Visual (Art Director)

- [ ] Banner evento en plaza legible
- [ ] Calendario UI claro (día, tipo, recompensa)
- [ ] Countdown visible pre-evento

## Regresión

- [ ] Evento cancelado devuelve entry fee (alpha rule)
- [ ] No solapamiento de 2 races críticas simultáneas
