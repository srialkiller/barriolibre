## Reglas del lore

1. **Tono:** infantil, cálido, latinoamericano — nunca violento ni criminal.
2. **Adultos:** toleran y supervisan desde la distancia; no son enemigos.
3. **Vehículos:** caseros, reciclados, nunca de fábrica ni ilegales.
4. **Carreras:** fiesta del barrio, no pandillismo.
5. **Filtro:** si un lore contradice GAME_IDENTITY → se reescribe el lore, no el ADN.

## Uso por agentes

- **World Designer** — layout coherente con adultos y POIs
- **Race Designer** — torneos y permisos
- **Creative Director** — validar narrativa emergente
- **Game Designer** — misiones y NPCs
